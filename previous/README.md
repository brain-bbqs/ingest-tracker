# Previous versions

Variations on this dashboard have existed over the last year to help keep track of the status of each project.

The `EMBER_Ingest_Tracker.csv` was an exported Slack table from a private APL channel, developed by Rahul Hingorani (@rhingo).

An early prototype of the KanBan board was also built by Rahul on the private GitHub Project found at [aplbrain/projects/23](https://github.com/orgs/aplbrain/projects/23).

Other internal reporting remains internal to APL Google Drive or Box, contributed by Nicole Guittari (@guittari), Neha Thomas (@nehatk17), Grace Bezold (@gbezold1), and Erik Johnson (@erikjohnson24).
